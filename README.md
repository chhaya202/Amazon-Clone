# Amazon Clone

A front-end clone of the Amazon homepage built using HTML and CSS.

## Features

* Amazon-style navigation bar
* Search bar design
* Product categories section
* Hero banner section
* Footer section
* Responsive and user-friendly layout

## Technologies Used

* HTML5
* CSS3

## Project Description

This project recreates the front page of the Amazon website using HTML and CSS. It was developed to practice web design, page layout, styling, and responsive user interface development.
